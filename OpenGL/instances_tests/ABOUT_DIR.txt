
Что тут собрано:
----------------

В этой папке собираются разные варианты организации логики работы 3D движка
для поиска максимально эффективной и быстрой реализации.

  3D сцена на которой построена поверхность в форме квадратного
текстурированного поля 1000*1000 прямоугольников
Каждая вершина имеет атрибуты
  - 3D координату,
  - 3D вектор нормали,
  - RGBA цвет,
  - UV координаты текстуры.
На прямоугольник по 2 треугольника: 6 млн. вершин без индексирования,
и 4 млн. вершин при использовании индексного буфера

1.plain_quads   (WIN-NV: 330fps, Lin-Intl: 27fps)
-------------------------------------------------
  Один общий буфер для всех атрибутов.

2.plain_quads   (WIN-NV: 435fps, Lin-Intl: 26fps)
-------------------------------------------------
  Для каждого из атрибутов создан отдельный буфер.

3.gl_triangle_strip (WIN-NV: 549fps, Lin-Intl: 36fps  )
-------------------------------------------------
  Общий буфер для атрибутов + индексный буфер GL_TRIANGLES_STRIP.

3.plain_quads   (WIN-NV: 470fps, Lin-Intl: 36fps)
-------------------------------------------------
  Общий буфер для атрибутов + индексный буфер GL_TRIANGLES.

4.plain_quads   (WIN-NV: 680fps, Lin-Intl: 34fps)
-------------------------------------------------
  Раздельные буферы для каждого из атрибутов + индексы.

5.inst          (WIN-NV: 205fps, Lin-Intl: 37fps)
-------------------------------------------------
  Копирующий рендер объекта: 6 вершин/2 треугольника GL_TRIANGLES
  Раздельные буферы для каждого из атрибутов

6.inst          (WIN-NV: 218fps, Lin-Intl: 51fps)
-------------------------------------------------
  Копирующий рендер объекта: 4 вершины/2 треугольника GL_TRIANGLE_STRIP
  Раздельные буферы для каждого из атрибутов

7.inst          (WIN-NV: 215fps, Lin-Intl: 50fps)
-------------------------------------------------
  Копирующий рендер объекта: 4 вершины/2 треугольника GL_TRIANGLE_STRIP
  Общий буфер атрибутов, минимизированый по 3 значения на атрибут,
  соответственно - изменения размерности входных переменных вершинного шейдера.
