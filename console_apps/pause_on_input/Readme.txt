
Тестирование вывода на консоль с паузой.

Для работы с консольным/терминальным окном из двх процессов одновременно.
Пример демонстрирует возможность приостановить вывод текста из одного процесса
на время ввода команды с клавиатуры в другом процессе, чтобы не мешать
оператору набирать команду.

Вывод из приостановленного потока на экран возобновляется по приему с
клавиатуры сивола перевода каретки или по установленному таймауту - чтобы
не заблокировать полностью вывод из первого процесса.
